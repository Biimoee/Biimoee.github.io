Les bases du CSS

Voici quelques notions de base du CSS

Le fond

Appliquer une couleur de fond

body { background-color: red; }

Appliquer une image de fond

body { background-image: url(nom_de_l_image.png) }


Le texte

Changer la couleur du texte

* { color: blue; }

Changer la police du texte

* { font-family: Verdana, Geneva, Tahoma, sans-serif; }

Modifier la taille du texte

* {font-size: 150%; }

Divers

Appliquer une marge

/*Partout*/
    body { margin: 50px; }
/*A gauche*/
    body { margin-left: 50px; }




Les principales balises HTML

Insérer une image

On utilise la balise <img> pour insérer une image dans une page HTML. C'est une balise auto fermante, c'est-à-dire qu'il n'y a pas besoin de la refermer.

Il existe de nombreux attributs qui peuvent se mettre dans cette balise. Certains sont obligatoires, d'autres non.
L'attribut <src> dit à la page web à afficher.
L'attribut <alt> associe un texte à cette image. Une voix peut par exemple lire ce texte pour les personnes malvoyantes. Ce texte s'affiche ausssi quand l'image ne peut pas s'afficher (quand le fichier est corrompu par exemple).
Je n'ai cité que les deux attributs obligatoires, il en existe beaucoup d'autres.

<img src="images/mon_image.jpg" alt="Une image">

Le retour à la ligne

<br>

Les titres

<h1>Titre 1</h1>
<h2>Titre 2</h2>




Les tableaux

La balise <table> </table> permet d'indiquer le début et la fin d'un tableau. 

<table> </table>

Voici deux nouvelles balises :

  • La balise <tr> </tr> indique le début et la fin d'une ligne du tableau
  • La balise <td> </td> indique le début et la fin du contenu d'une cellule

En HTML, un tableau se construit ligne par ligne. Dans chaque ligne (<tr>), on indique le contenu des différentes cellules (<td>).

Exemple

<table>
        <tr>
            <td>A</td>
            <td>B</td>
            <td>C</td>
        </tr>
        <tr>
            <td>1</td>
            <td>2</td>
            <td>3</td>
        </tr>
     </table>
     <style>td{border: 1px solid black;}</style>
