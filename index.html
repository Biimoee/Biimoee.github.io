Les bases du CSS

Voici quelques notions de base du CSS

Le fond

Appliquer une couleur de fond

body { background-color: red; }

Appliquer une image de fond

body { background-image: url(nom_de_l_image.png) }


Le texte

Changer la couleur du texte

* { color: blue; }

Changer la police du texte

* { font-family: Verdana, Geneva, Tahoma, sans-serif; }

Modifier la taille du texte

* {font-size: 150%; }

Divers

Appliquer une marge

/*Partout*/
    body { margin: 50px; }
/*A gauche*/
    body { margin-left: 50px; }




Les principales balises HTML

Insérer une image

On utilise la balise pour insérer une image dans une page HTML. C'est une balise auto fermante, c'est-à-dire qu'il n'y a pas besoin de la refermer.

Il existe de nombreux attributs qui peuvent se mettre dans cette balise. Certains sont obligatoires, d'autres non.
L'attribut dit à la page web à afficher.
L'attribut associe un texte à cette image. Une voix peut par exemple lire ce texte pour les personnes malvoyantes. Ce texte s'affiche ausssi quand l'image ne peut pas s'afficher (quand le fichier est corrompu par exemple).
Je n'ai cité que les deux attributs obligatoires, il en existe beaucoup d'autres.

Les tableaux

La balise

permet d'indiquer le début et la fin d'un tableau.

Voici deux nouvelles balises :

  • La balise indique le début et la fin d'une ligne du tableau
  • La balise indique le début et la fin du contenu d'une cellule

En HTML, un tableau se construit ligne par ligne. Dans chaque ligne (), on indique le contenu des différentes cellules ().

Exemple


        

            A
            B
            C
        

        

            1
            2
            3

Résulat

A B C
1 2 3
